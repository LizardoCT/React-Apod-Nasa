## Run Locally

Clone the project

```bash
  git clone https://github.com/LizardoCT/React-Apod-Nasa
```

Go to the project directory

```bash
  cd React-Apod-Nasa
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
